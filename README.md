# Password Strength Checker & Generator (Java)

## Description
A Java console application that checks the strength of a password
and generates secure random passwords.

## Concepts Used
- Core Java
- Regular Expressions
- Random class
- OOP principles

## Features
- Password strength analysis
- Secure password generation
- Menu-driven console UI

## How to Run
javac *.java
java Main
